🛠️ 指令運用與技術說明

本學期透過 p5.js 學習了程式繪圖的核心邏輯，並將其運用在此海底主題畫廊中：

  • 物件導向程式設計 (Class)： 使用 class Seaweed 封裝海草的屬性與行為，透過 constructor 初始化位置，並利用陣列管理多個海草物件。
  • 數學三角函數： 運用 sin() 函數結合 frameCount，計算出規律的海浪擺動與魚尾撥動效果。
  • 互動偵測： 使用 dist() 指令計算滑鼠與海草段落的距離，實現滑鼠靠近時海草自動避開的互動反饋。
  • 實例模式 (Instance Mode)： 為了在同一個網頁中運行多個不同的 p5 畫布（背景、作業一、作業二），運用了 new p5() 封裝技術，避免全域變數衝突。
  • 動態響應： 透過 windowResized() 與相對單位計算，確保海底環境在不同視窗大小下都能自動重新佈局。
